# Sprint 1 — MVP Manual Deployment

## Objective
Deploy a working Flask application connected to MariaDB on 2 VMs

## Scope
- VM provisioning (manual)
- MariaDB setup
- Flask app deployment
- Network connectivity
- Basic documentation

## Demo
- Web form accessible
- Data inserted in DB
- Verification via SQL

## Definition of Done
- [x] App accessible via browser
- [x] Data insertion works
- [x] Data visible in DB
- [x] All steps understood and documented

## Retrospective

### What went well
- Clear separation app / db
- MVP delivered quickly

### What can be improved
- Manual setup is time-consuming
- Configuration not reproducible

### Next focus
- Introduce Ansible for configuration automation
